1. Кумулятивная флоу-диаграмма (Cumulative Flow Diagram)

Показывает накопленное количество задач в каждом состоянии во времени. Помогает оценить стабильность потока и выявить узкие места.

6. Другие рекомендуемые визуальные отчёты

  • Диаграмма Ганта — временная шкала задач по спринтам/версиям
  • Velocity Chart — скорость команды (завершённые SP/итерации по спринтам)
  • Lead/Cycle Time — время от создания до завершения задачи
  • Распределение по подсистемам — осмотры, типы МИ, отчёты и т.д.
  • Control Chart — время от «In Progress» до «Done» по задачам
  • Ageing Report — сколько задач «зависли» в каждом состоянии
  • Throughput — количество завершённых задач в неделю
